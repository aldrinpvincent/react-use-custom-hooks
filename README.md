# react-use-custom-hooks

A collection of react custom hooks

# Installation

```bash
npm install --save react-use-custom-hooks
yarn add react-use-custom-hooks
```

# Usage

```ts
import { useSafeState } from 'react-use-custom-hooks';
```

# Hooks list

1. useDebounce
2. useSafeState
3. useIsMounted
4. usePrevious
