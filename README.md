# react-use-custom-hooks

A collection of react custom hooks in typescript

# Installation

```bash
npm install --save react-use-custom-hooks
yarn add react-use-custom-hooks
```

# Usage

```ts
import { useDebounce } from 'react-use-custom-hooks';
```
